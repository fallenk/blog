# 题目

https://leetcode-cn.com/problems/is-graph-bipartite



# 解法



## 心得

题意: 读题的技巧，分析题目的要求是什么，想要求出的答案是什么样

方法：对于图的一类的问题，通常的解决办法都是 DFS， BFS， 或者并查集。

````java
class Solution {
    private static final int UNCOLORED = 0;
    private static final int RED = 1;
    private static final int GREEN = 2;
    private int[] colors;
    private boolean valid;
    public boolean isBipartite(int[][] graph) {
        if (graph == null) {
            return false;
        }
         
        int len = graph.length;
        colors = new int[len]; // 记录颜色
        valid = true; 
        Arrays.fill(colors, UNCOLORED); 
        
        for (int i = 0; i < len && valid; i++) {
            if (colors[i] == UNCOLORED) {
                dfs(i, RED, graph);
            }
        }
        return valid;
    }
    /**
     * 迭代，获取当前节点颜色，染下一个节点
     * @param i
     * @param red2
     * @param graph
     */
    private void dfs(int i, int color, int[][] graph) {
        if (i > graph.length - 1 ) {
            return;
        }
        colors[i] = color;
        int colorNei = color == RED ? GREEN : RED; // 相邻颜色
        for (int neighbor : graph[i]) {
            if (colors[neighbor] != UNCOLORED && colors[neighbor] != colorNei) {
                valid = false;
                return;
            }
            // 没染色 -》colored
            if (colors[neighbor] == UNCOLORED) {
                dfs(neighbor, colorNei, graph);
            }
        }
    }
}
````
